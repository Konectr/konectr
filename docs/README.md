```
docs/
├── README.md              # This file

│   └── YYYY-MM-DD.md      # Daily development logs
```

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **i18n**: next-intl (8 languages)
- **TypeScript**: Strict mode enabled
